## Demos Included

### Liquid Glass
- Glass Cards
- Glass Navigation
- Glass Tab Bar
- Glass Sheets
- Glass Alerts

### Foundation Models
- Text Generation
- Summarization
- Translation
- Code Completion
- Image Captioning

### App Intents 2.0
- Custom Shortcuts
- Siri Integration
- Automation Triggers

### Other Features
- Spatial Audio
- Vision APIs
- New SwiftUI Views
- Enhanced Animations

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/muhittincamdali/SwiftUI-iOS26-Showcase.git", from: "1.0.0")
]
```

### CocoaPods

```ruby
pod 'SwiftUIiOS26Showcase', '~> 1.0'
```
